# Cafetaria de Sluis

This is the website for cafetaria de sluis.

## Features

- New website
- Able to view contact details
- Able to view our menu
- Able to send us messages

## Upcoming Features

- Allergy list

## Tech Stack

This website is built with the following technologies:

- Tailwind CSS
- React
- Node
- Vite
